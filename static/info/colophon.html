About this Site

This site was created by a group of local residents of St Margarets, Middlesex. Our mission is simple — help foster a stronger community identity. There is no hidden agenda, no political affiliations and to date, all the work is volunteer.

Who is Involved

The site was created by Peter Mahnke, with input from Ann Murray, Angela Mahnke and Simon Chapman.

How can you help?

Well, as you might have guessed, this site is no one's day job. So if anyone has leads, ideas, additional events or wants to help contribute in other ways, please email us.

How was this site built?

It was built using mostly free, or very inexpensive tools:

  • the core of the site is based on a popular bloging tool called Movable Type
    • we also are using MT-Textile and SmartyPants plugins if you really care to know
  • forums are done with the customs software
  • static pages (like this one) are built using a custom made CMS (its ok if you don't know what that means)
  • the train timetables, events, polls, local information/directory, newsletter sign-up are based on a custom Perl script

If you are interested, we are maintaining a change log.

The site doesn't look good in my Browser

The site is built with and validated against established web standards. Your problem might be that your browser is older and doesn't render web standards well. We can recommend switching to Firefox or “Google Chrome”: https://www.google.com/intl/en/chrome/browser/

Who is paying for the site?

Currently the site is being supported by Peter Mahnke, but over time, I hope that donations and advertising help offset some of the costs. If you are interested in donating time or money or interested in advertising, please contact us.
